React Roadmap Checklist

1. Prerequisites

  • HTML & CSS (Flexbox, Grid, Media Queries)
  • Modern JavaScript (ES6+)
  • Git and GitHub Basics
  • npm or yarn

2. React Basics

  • JSX
  • Functional Components
  • Props and State
  • Event Handling
  • Lists & Keys
  • Conditional Rendering

3. React Hooks

  • useState, useEffect
  • useRef
  • useContext, useReducer
  • useMemo, useCallback

4. Routing

  • React Router Basics
  • BrowserRouter, Route, Link
  • Navigation with useNavigate

5. State Management

  • Context API
  • Redux Toolkit / Zustand
  • React Query / SWR

6. Styling

  • CSS Modules
  • Styled Components / Tailwind CSS
  • Responsive Design

7. Testing

  • Unit Testing (Jest)
  • React Testing Library
  • End-to-End (Cypress)

8. Deployment

  • Deploy on Vercel / Netlify / GitHub Pages
  • Learn CI/CD Basics

9. Advanced Concepts

  • Code Splitting, Lazy Loading
  • Error Boundaries
  • Custom Hooks
  • Next.js & SSR
  • Progressive Web Apps (PWA)

10. Build Projects

  • Todo List App
  • Weather App with API
  • Markdown Blog
  • E-commerce App with Auth
