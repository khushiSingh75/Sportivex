The display property specifies the display behavior (the type of rendering box) of an element.

Block elements: div, p, h, table, section, ul, ol, article
  1. Always starts with a new line.
  2. Starts from extreme left to extreme right(take full width).
  3. We can set custom width and height.
  4. We can set custom margin and padding.


Box-sizing: border-box;
  Without this the actual width of the box or element increases.
  This is a CSS reset to remove default margin and padding from all elements.


Inline elements: span, a, img, button, input
  1. Try to adjust in same line, if multiple elements and also if required width/space
  2. Width = content width
  3. We can not apply custom height nd width and margin(only in horizontal).


display: block/inline/inline-block
inline-block -- behave like inline but we can set custom width/height nd margin/padding.
